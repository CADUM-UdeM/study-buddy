# Fonts

JetBrains Mono Nerd Font is loaded by the app from this folder.

To install it, run from the project root:

```bash
node scripts/download-jetbrains-mono-nerd-font.js
```

Or download manually from [Nerd Fonts](https://www.nerdfonts.com/font-downloads) (JetBrains Mono) and place `JetBrainsMonoNerdFont-Regular.ttf` here.
